# 🌍 Vihangam: Intelligent Tour Routing Engine

## 📌 Project Description
Vihangam is an intelligent travel route optimization system developed using Python. It uses **Dijkstra’s Algorithm** to find the shortest path between cities and provides an interactive map-based visualization of routes.

This project demonstrates the real-world application of **Design and Analysis of Algorithms (DAA)** in navigation, logistics, and transportation systems.

---

## 🚀 Features
- Shortest path calculation using Dijkstra’s Algorithm  
- Graph-based representation of cities and distances  
- Interactive user interface using Streamlit  
- Map visualization using Folium  
- Displays:
  - Shortest path (step-by-step)
  - Total distance
  - Route on map  
- Supports multiple interconnected cities (Maharashtra dataset)

## 🧠 Algorithm Used
### Dijkstra’s Algorithm
- A greedy algorithm used to find the shortest path in a weighted graph  
- Uses a priority queue to select the minimum distance node  
- Time Complexity: **O(E log V)**
